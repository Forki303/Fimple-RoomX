# Fimple RoomX




Простой консольный чат на Python с сервером и клиентом через TCP sockets.

---

## 📦 Возможности

- Общение в реальном времени
- Подключение нескольких пользователей
- Никнеймы пользователей
- Пароль для входа в комнату
- Оповещения о подключении/выходе


---

## ⚙️ Требования

- Python 3.8+
- Git (без распаковки)



---

## 📥 Установка

### 1. Клонировать репозиторий
```bash
git clone hhttps://github.com/Forki303/Fimple-RoomX.git
cd Fimple-RoomX
```


### 2. Или скачать вручную

Скачать ZIP и распаковать архив.

---

## 🚀 Запуск
🖥️ Сервер (host):
python server.py

По умолчанию:
```
IP: 0.0.0.0
Порт: 25565
Пароль комнаты: 1234
```
💻 Клиент:
python client.py

Далее:

Введите IP сервера <br>
Введите порт (например 25565) <br>
Введите пароль <br>
Введите ник <br>


## ⚠️ ВАЖНО:

Чтобы другие пользователи могли подключиться:

Нужно открыть порт 25565 на роутере (Port Forwarding) <br>
Или использовать VPN (например Hamachi / Radmin VPN) <br>
Или проброс через ngrok

---

## 🔐 Безопасность
Пароль комнаты используется только на уровне приложения <br>
Не предназначено для публичных серверов <br>
Нет шифрования данных <br>
